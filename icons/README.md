# Icons

このフォルダにはPWAアプリのアイコンファイルが必要です。

## アイコンの作成方法

### 方法1: create-icons.htmlを使用
1. `create-icons.html` をブラウザで開く
2. 自動的に `icon-192.png` と `icon-512.png` がダウンロードされます
3. ダウンロードしたファイルをこのフォルダに配置

### 方法2: オンラインツールを使用
以下のようなツールで192x192と512x512のPNG画像を作成:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

### 方法3: 画像編集ソフトを使用
お好きな画像編集ソフト(Photoshop, GIMP, Canvaなど)で以下のサイズのPNG画像を作成:
- `icon-192.png` (192x192px)
- `icon-512.png` (512x512px)

デザインのヒント:
- バナナ🍌をモチーフにした黄色い背景
- グラデーション: #FFA500 → #FFD700
- シンプルで認識しやすいデザイン
